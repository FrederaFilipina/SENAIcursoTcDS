Método http de comunicação
>> O método de comunicação HTTP é baseado em um modelo de solicitação-resposta entre um cliente
e um servidor, sendo o HTTP (Hypertext Transfer Protocol) o protocolo que define as regras para
essa comunicação. O cliente (como um navegador) envia uma requisição e o servidor responde com
um cabeçalho e, opcionalmente, um corpo da mensagem. Para especificar a ação desejada, utilizam-se
métodos como GET para buscar dados, POST para enviar dados, PUT para atualizar e DELETE para remover
um recurso.
